    • Questions

    1. One should use Emmet since it is a useful tool, that once learned can increase the speed at which one can generate HTML code.
    2. The useful abbreviations can be found in the cheat-sheet.
    3. The syntax to generate a paragraph with the class "test" is: 'p.text'
    4. This syntax that generates an element with a child inside of it is: 'div>p' or simply '>p'. Since '>' is used to indicate a child of the former.
    5. To iteratively create stuff, one can use the sign '*'. So, three elements with the same class name can be created like this: '.class-name*3'
